## 二、商店定价

### 2.1 收购价（玩家→商店）

商人以物品价值的 **90%** 收购。

```
收购价 = 物品价值 × 0.9

例：
  钻石(600) → 商店收购 540 金币
  AKM(550)  → 商店收购 495 金币
  遗物圣器(3000) → 商店收购 2700 金币
```

### 2.2 售价（商店→玩家）

商店只售卖四类物品：**武器、背包、防具、药品**。

售价 = 物品价值 × **3.0**

```
例：
  Glock(200) → 商店售价 600 金币
  防弹头盔(200) → 商店售价 600 金币
  AKM(550) → 商店售价 1650 金币
  复合陶瓷甲(1200) → 商店售价 3600 金币
```

### 2.3 商店售卖品类

| 品类 | 售卖 | 说明 |
|------|------|------|
| 武器 | ✅ | 手枪到狙击全系列 |
| 防具 | ✅ | 头盔+护甲 |
| 背包 | ✅ | 小到大 |
| 药品 | ✅ | 绷带/急救包/手术包/止痛药/肾上腺素 |
| 弹药 | ❌ | 只能在地牢里搜 |
| 战利品 | ❌ | 只能在地牢里搜，也只能卖给商店 |
| 垃圾 | ❌ | 同上 |

## 五、经济平衡测算

### 5.1 新手期（1~5层）

```
收入：300~600/局
支出：0~250/局（轻装）
净赚：50~600/局

目标：
  攒够 800 金币 → 买 MP5
  攒够 1210 金币 → 买 AKM
  
需要：约 3~6 局（存活前提下）
```

新手期经济紧张但不是太难，几局就能买得起主力武器。

### 5.2 中期（6~10层）

```
收入：1500~3500/局
支出：500~1500/局（中装）
净赚：0~3000/局

目标：
  攒够 3000 → 买重型防弹衣
  攒够 5000 → 升级仓库到 12×10
  
需要：约 2~5 局
```

中期收入开始稳定，可以开始积累财富。

### 5.3 后期（11~15层）

```
收入：6000~14000/局
支出：2000~5000/局
净赚：1000~12000/局

目标：
  攒够 8000 → 买 AUG + 配件
  攒够 20000 → 升级仓库到 14×12
  
需要：约 2~4 局
```

### 5.4 终局（16~20层）

```
收入：25000~120000/局
支出：5000~15000/局
净赚：10000~105000/局

目标：所有装备拉满，刷 20 层 Boss
```

## 六、经济循环图

```
出战(花钱买装备) → 入地牢 → 搜刮/击杀
       ↓                              ↓
  阵亡(装备损失)                  存活(带回战利品)
       ↓                              ↓
  仓库不动                       战利品入仓库
                                    ↓
                              卖给商人(获得金币)
                                    ↓
                        购买更贵装备(扩大再生产)
```

### 关键平衡点

| 指标 | 数值 | 说明 |
|------|------|------|
| 单局最大支出 | ~15000 | 20层满配成本 |
| 单局最大收入 | ~120000 | 20层Boss全拿 |
| 风险回报比 | 1:8 | 花1万能赚8万，阵亡亏1万 |
| 仓库满级成本 | 75000 | 需要很多局的积累 |
| 收支平衡存活率 | 20% | 只要存活率>20%，长期就赚钱 |
| 破产保护 | 裸装Glock免费 | 永远可以0成本出战 |

## 七、实现要点

```javascript
// 物品出售给商店
function sellItem(item) {
  const price = Math.floor((item.value || 0) * 0.9);
  player.money += price;
  return price;
}

// 从商店购买物品（仅武器、防具、背包、药品）
function buyItem(item) {
  // 商店不出售弹药、战利品和垃圾
  if (!['weapon','armor','helmet','backpack','consumable'].includes(item.type)) return false;
  const price = Math.floor((item.value || 0) * 3.0);
  if (player.money < price) return false;
  player.money -= price;
  return true;
}

// 按层数调整价值倍率
function getLootValueMultiplier(floor) {
  if (floor <= 4)  return 1.0;
  if (floor <= 9)  return 1.5;
  if (floor <= 14) return 2.5;
  return 4.0;
}
```
