## 二、区域划分

### 2.1 顶部状态栏（Top Left + Top Right）

**左上：玩家信息面板**

```
┌──────────────────────┐
│  🐱 修勾   Lv.6      │  ← 头像 + 等级
│  🔴 红币  10         │  ← 付费货币
│  🟡 金币  5,400,000  │  ← 免费货币
│  ⬛⬛⬛⬛⬛⬛⬛⬛⬜⬜ 351/432 │  ← 经验条
└──────────────────────┘
```

样式：
- 背景：rgba(15,20,25,0.85)，圆角 8px
- 内边距：12px
- 字体：白色/金色，等宽数字

**右上：功能入口**

```
┌────┬────┬────┬────┐
│ 🏆 │ 🛒 │ ✉️ │ ⚙️ │
│排行│商城│邮件│设置│
└────┴────┴────┴────┘
```

样式：
- 图标按钮，48×48px
- 悬停时发光边框

### 2.2 左侧导航栏（Left Sidebar）

```
┌──────┐
│ ✅ 任务 │
│ 💪 天赋 │
│ 🐾 宠物 │
│ 🔨 熔炉 │
│ 📦 收藏 │
└──────┘
```

样式：
- 垂直排列，固定宽度
- 图标+文字，选中态高亮
- 背景半透明，与主区域分离

### 2.3 中央背景区域（Center）

```
背景：深色底 + 武器剪影重复排列（半透明，作为纹理）
中央：一个小像素角色（猫持枪站立）

所有 UI 面板叠在背景之上
```

渲染方式：

```javascript
function renderBackground(ctx) {
  // 深色基底
  ctx.fillStyle = '#0f1419';
  ctx.fillRect(0, 0, W, H);
  
  // 武器剪影纹理（重复排列，半透明）
  ctx.globalAlpha = 0.06;
  const weapons = ['🔫', '🗡️', '🔧'];
  for (let y = 0; y < H; y += 80) {
    for (let x = 0; x < W; x += 80) {
      ctx.font = '40px Arial';
      ctx.fillText(weapons[(x + y) % 3], x, y);
    }
  }
  ctx.globalAlpha = 1;
}
```

### 2.4 底部导航栏（Bottom Dock）

```
┌────────┬────────┬────────┬────────┬────────┐
│ 🎒    │ 🛍️    │ 🔨    │ 🔧    │ 🕶️    │
│ 背包   │ 商店   │ 制作   │ 改装   │ 黑商   │
└────────┴────────┴────────┴────────┴────────┘
```

样式：
- 5 个按钮横向等距排列
- 选中态：下划线或高亮色
- 背景：rgba(15,20,25,0.9)，顶部细分割线

### 2.5 主操作按钮（Bottom Right）

```
┌──────────────────┐
│  🗺️  选择地图     │  ← 最大的按钮，吸引点击
└──────────────────┘
```

样式：
- 大号，圆角
- 主色 #00f0ff 或 #e94560
- 悬停发光
- 位置固定在右下角，不随滚动移动

## 五、实现要点

```javascript
// 主界面渲染
function renderMainMenu() {
  ctx.clearRect(0, 0, W, H);
  
  // 1. 背景
  renderBackground(ctx);
  
  // 2. 左上玩家信息
  renderPlayerPanel(ctx);    // x:16, y:16
  
  // 3. 右上方功能入口
  renderTopIcons(ctx);       // x:W-200, y:16
  
  // 4. 左侧导航
  renderSidebar(ctx);        // x:16, y:120
  
  // 5. 中央角色
  renderCharacter(ctx);      // center
  
  // 6. 底部导航栏
  renderBottomDock(ctx);     // y:H-60
  
  // 7. 主操作按钮
  renderMainButton(ctx);     // x:W-200, y:H-150
}
```
