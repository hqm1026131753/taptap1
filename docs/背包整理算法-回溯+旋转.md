# 网格背包整理算法：回溯 + 旋转

网格背包的整理本质是二维矩形装箱，物品可以旋转 90 度（交换宽高）。用回溯法在有限格子数的背包里找最优排列。

---

## 核心策略

```
1. 取出背包中所有物品
2. 按面积从大到小排序（大件优先）
3. 对每个物品尝试所有空位 + 两个朝向（原方向/旋转90度）
4. 一种排列尝试到底（DFS），放不下最后一个物品时回溯
5. 找到完全放置方案后输出
```

---

## 完整代码

```lua
-- ============================================================
-- BackpackSorter.lua
-- 网格背包整理：回溯 + 旋转
-- 使用场景：玩家点击"一键整理"按钮时调用
-- ============================================================

local BackpackSorter = {}

-- 深拷贝网格
local function cloneGrid(grid)
    local g = {}
    for r = 1, #grid do
        g[r] = {}
        for c = 1, #grid[r] do
            g[r][c] = grid[r][c]
        end
    end
    return g
end

-- 检查某个位置能否放下物品
local function canPlace(grid, row, col, w, h)
    if row + h - 1 > #grid or col + w - 1 > #grid[1] then
        return false
    end
    for r = row, row + h - 1 do
        for c = col, col + w - 1 do
            if grid[r][c] ~= nil then
                return false
            end
        end
    end
    return true
end

-- 放置物品
local function placeItem(grid, id, row, col, w, h)
    for r = row, row + h - 1 do
        for c = col, col + w - 1 do
            grid[r][c] = id
        end
    end
end

-- 移除物品
local function removeItem(grid, row, col, w, h)
    for r = row, row + h - 1 do
        for c = col, col + w - 1 do
            grid[r][c] = nil
        end
    end
end

-- ============================================================
-- 回溯放置（核心算法）
-- ============================================================
-- 逐个尝试物品的每个空位和每个朝向
-- 如果当前物品的所有放置尝试失败，回溯到上一个物品换位置

function BackpackSorter.Sort(backpack)
    if not backpack or not backpack.grid then return end

    local grid = backpack.grid        -- 二维表 [row][col]
    local rows = #grid
    local cols = #grid[1]

    -- 第 1 步：取出所有物品
    local items = {}
    for r = 1, rows do
        for c = 1, cols do
            local id = grid[r][c]
            if id and not items[id] then
                local item = backpack.items[id]
                if item then
                    items[id] = {
                        id = id,
                        w = item.gridW or item.w,         -- 占格宽
                        h = item.gridH or item.h,         -- 占格高
                        value = item.value or 0,
                        category = item.category or "misc",
                    }
                end
            end
        end
    end

    -- 第 2 步：转为有序列表，按面积降序
    local itemList = {}
    for id, data in pairs(items) do
        data.area = data.w * data.h
        table.insert(itemList, data)
    end
    table.sort(itemList, function(a, b)
        if a.area ~= b.area then return a.area > b.area end
        return a.value > b.value
    end)

    -- 第 3 步：清空格子
    for r = 1, rows do
        for c = 1, cols do
            grid[r][c] = nil
        end
    end

    -- 第 4 步：回溯放置
    local bestGrid = nil
    local bestPlaced = 0
    local bestValue = 0
    local searchedNodes = 0

    local function backtrack(index, placedCount, totalValue)
        searchedNodes = searchedNodes + 1

        -- 剪枝：搜索节点过多时停止（防止卡死）
        if searchedNodes > 50000 then
            return
        end

        if index > #itemList then
            -- 所有物品都放完了，更新最优解
            if placedCount > bestPlaced or
               (placedCount == bestPlaced and totalValue > bestValue) then
                bestPlaced = placedCount
                bestValue = totalValue
                bestGrid = cloneGrid(grid)
            end
            return
        end

        local item = itemList[index]
        local orientations = {
            { w = item.w, h = item.h },
            { w = item.h, h = item.w },  -- 旋转 90 度
        }

        -- 如果宽高相同，不重复尝试旋转
        if item.w == item.h then
            orientations = { { w = item.w, h = item.h } }
        end

        -- 尝试每个空位 × 每个朝向
        for _, orient in ipairs(orientations) do
            for r = 1, rows - orient.h + 1 do
                for c = 1, cols - orient.w + 1 do
                    if canPlace(grid, r, c, orient.w, orient.h) then
                        placeItem(grid, item.id, r, c, orient.w, orient.h)
                        backtrack(index + 1, placedCount + 1, totalValue + (item.value or 0))
                        removeItem(grid, r, c, orient.w, orient.h)

                        -- 如果已经找到最优解（所有物品全放），直接返回
                        if bestPlaced == #itemList then
                            return
                        end
                        -- 搜索上限
                        if searchedNodes >= 50000 then
                            return
                        end
                    end
                end
            end
        end
    end

    backtrack(1, 0, 0)

    -- 第 5 步：应用最优解
    if bestGrid then
        for r = 1, rows do
            for c = 1, cols do
                grid[r][c] = bestGrid[r][c]
            end
        end
    end

    return bestPlaced
end

return BackpackSorter
```

---

## 算法分析

| 指标 | 值 |
|------|-----|
| 搜索上限 | 50000 节点（防止卡死） |
| 剪枝策略 | 大件优先 + 面积降序 |
| 旋转尝试 | 宽高不同时尝试两个朝向 |
| 最优标准 | 先比放置数量，再比总价值 |
| 容量 5×8 背包 | 约 10~15 个物品时在毫秒级完成 |

## 集成方式

```lua
local Sorter = require("scripts.BackpackSorter")

-- 玩家点击"一键整理"时：
function OnSortBackpack()
    local placed = Sorter.Sort(player.backpack)
    print("整理了 " .. placed .. " 件物品")
end
```

## 优化建议

1. **类别分区**——在回溯前先把物品按类别分组，每个类别在背包中划一块区域，减少跨类搜索
2. **格数预检**——如果物品总面积 > 背包总格数，直接跳过不可能放得下的物品（先扔垃圾再整理）
3. **渐进整理**——不是每次全部打乱重排，而是只在玩家手动放不下时才触发
