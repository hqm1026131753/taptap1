# 修勾大逃亡 — 音效与BGM设计

## 一、BGM 设计

### 1.1 主菜单 BGM

```
风格：Dark Ambient + 极简电子
时长：循环 60s
调性：D 小调，80 BPM

结构：
  0:00~0:08  || 低沉的次低音脉冲（sub-bass pulse），每2秒一次
  0:08~0:16  || 加入远距离的合成器Pad（C5-D5 缓慢扫弦）
  0:16~0:30  || 加入微弱的打击乐——像远处金属管道滴水的声音
  0:30~0:45  || 一个单音旋律动机（D5-F5-G5，高八度，带混响）
  0:45~0:60  || 回归低音脉冲，淡化结束

情绪：孤独、警觉、未知——玩家还没进入地牢，但已经感受到压力。
```

### 1.2 地牢探索 BGM

```
风格：Dark Ambient + 持续的紧张感
时长：循环 90s
调性：无调性，以氛围为主

分层设计：
  环境层：    持续的沙沙声（cave drone）+ 偶尔的水滴声（每10~20s随机）
  节奏层：    极简的打击乐——每4秒一个低音鼓点（像心跳）
  变化层：    敌人靠近时频率加快（距离<200px时鼓点变2倍速）
  高潮层：    战斗时触发——叠加急促的电子鼓（drum & bass风格, 140 BPM）

动态切换：
  探索态（无敌人）：环境层 + 节奏层（慢速）
  警戒态（敌人附近）：节奏层加速
  战斗态：叠加高潮层
  战后恢复：30s 内逐渐回到探索态
```

### 1.3 Boss 战 BGM

```
风格：工业金属 + 电子交响
时长：循环 45s，持续到 Boss 死亡

结构：
  0:00~0:10  || 突然爆发的失真合成器音墙 + 低音鼓重击（fortissimo）
  0:10~0:20  || 加入高音失真旋律（tritone 音程，制造不和谐感）
  0:20~0:35  || 节奏加倍，鼓点密集，加入类似警报的周期性音效
  0:35~0:45  || 短暂回落，但低音持续，准备下一轮爆发

情绪：压迫感、绝望感、 adrenaline 飙升
```

### 1.4 撤离 BGM

```
风格：渐强的希望感
时长：15s（一次性播放，不循环）

结构：
  0:00~0:05  || 静默——只保留环境风声
  0:05~0:10  || 一个干净的合成器音（C5 长音），缓慢渐强
  0:10~0:15  || 加入第二个音（G5），形成五度和谐，伴随微弱的光芒感

情绪：如释重负、短暂的安全感
```

### 1.5 阵亡 BGM

```
风格：骤停 + 衰减
时长：8s

效果：
  0:00       || 所有声音瞬间切断（断层 silence，制造冲击）
  0.5~2s    || 低沉的心跳声（slow, 30 BPM）
  2~8s      || 声音逐渐被类似水下的低频噪音淹没，直至完全安静

情绪：戛然而止、失落、寂静
```

## 三、音效优先级与混音

### 3.1 层级

```
优先级 1（最重要的——不能遮罩）：
  武器射击 · 受伤痛呼 · Boss 登场 · 阵亡

优先级 2（重要——短暂音效）：
  换弹 · 拾取物品 · UI 确认 · 敌人死亡 · 受击

优先级 3（氛围——可被遮罩）：
  脚步声 · 环境水滴 · 远处金属声

优先级 4（背景——持续音）：
  BGM · 风声 · 机器嗡鸣
```

### 3.2 3D 空间音频

```
左右声道根据音源在屏幕上的位置偏移：
  音源在屏幕左侧 → 左声道更响
  音源在屏幕右侧 → 右声道更响
  音源在屏幕外 → 声音减弱 + 混响增强（暗示来自远处）
  音源距离 > 400px → 音量降至 30% + 增加低频衰减
```

## 五、实现参考

```javascript
// 音效管理器
class AudioManager {
  constructor() {
    this.sfx = {};
    this.bgm = null;
    this.masterVolume = 0.8;
    this.sfxVolume = 1.0;
    this.bgmVolume = 0.5;
  }

  // 预加载音效
  preload(sfxList) {
    for (const [name, path] of Object.entries(sfxList)) {
      const audio = new Audio(path);
      audio.preload = 'auto';
      this.sfx[name] = audio;
    }
  }

  // 播放一次性音效
  playSfx(name, options = {}) {
    const audio = this.sfx[name];
    if (!audio) return;
    const clone = audio.cloneNode();
    clone.volume = (options.volume || 1) * this.sfxVolume * this.masterVolume;
    clone.play();
  }

  // 播放 BGM（循环）
  playBgm(path) {
    if (this.bgm) this.bgm.pause();
    this.bgm = new Audio(path);
    this.bgm.loop = true;
    this.bgm.volume = this.bgmVolume * this.masterVolume;
    this.bgm.play();
  }

  // 3D 空间音效（基于屏幕位置）
  playSpatialSfx(name, screenX, screenY, options = {}) {
    const centerX = 400, centerY = 250;
    const dist = Math.hypot(screenX - centerX, screenY - centerY);
    const maxDist = 500;
    const pan = Math.max(-1, Math.min(1, (screenX - centerX) / centerX));
    const vol = Math.max(0, 1 - dist / maxDist);
    
    this.playSfx(name, {
      volume: options.volume * vol,
      pan: pan,
    });
  }

  // BGM 动态切换（探索→战斗）
  setCombatIntensity(intensity) {

    // 0 = 探索态，1 = 激烈战斗
    // 调整 BGM 的滤波/速度/叠加层
  }
}
```
