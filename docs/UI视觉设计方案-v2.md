# 修勾大逃亡 — UI 视觉设计方案 v2

> 风格定位：黑暗科技 + 玻璃拟态 + 冷光点缀。
> 融合科幻实验室监控界面的冷静与专业感，适配俯视角生存搜刮题材。

---

## 一、视觉基调

```
色彩公式：「深色基底 + 青/蓝高光 + 白字 + 红点」
质感核心：「深色玻璃拟态 + 科技发光」
布局哲学：「功能环绕，模块清晰，中心留焦」
情感基调：「专业、冷静、数据化、沉浸式」
```

---

## 二、色板

### 2.1 基础色

```
背景层     #0f1419      最深底色，用于全屏覆盖
面板层     #1a1f26      面板/卡片基底
玻璃面板   rgba(30,35,45,0.7)   半透明玻璃，带背景模糊
边框       rgba(0,240,255,0.15)  极细冷光边框
```

### 2.2 功能色

```
主高亮     #00f0ff      按钮激活、选中状态、关键数据
辅高亮     #80ffff      悬停态、数据图表、次要高亮
文字主色   #ffffff      标题、关键数值
文字辅色   rgba(255,255,255,0.7)   正文
文字弱色   rgba(255,255,255,0.4)   辅助信息、标签
警示色     #ff6b6b      警告、危险、低血量
成功色     #4ecca3      撤离安全、治疗中
稀有度紫   #a855f7      史诗物品边框
稀有度金   #ffd700      传说物品边框+发光
```

## 三、玻璃拟态规范

### 3.1 面板层级

```
层级0（最深）：  #0f1419                   全屏背景
层级1（面板）：  rgba(30,35,45,0.85)       主面板
层级2（子面板）：rgba(35,40,50,0.7)        子面板/卡片
层级3（悬浮）：  rgba(40,45,55,0.6)        悬浮/提示
```

所有面板使用 `backdrop-filter: blur(8px)` 模拟玻璃模糊。

### 3.2 发光参数

```
按钮边框发光：  0 0 6px rgba(0,240,255,0.3)
悬停态发光：    0 0 12px rgba(0,240,255,0.5)
激活态发光：    0 0 20px rgba(0,240,255,0.6)
传说物品发光：  0 0 15px rgba(255,215,0,0.5)
警示发光：      0 0 10px rgba(255,107,107,0.4)
```

### 3.3 边框规范

```
默认边框：  1px solid rgba(0,240,255,0.15)
悬停边框：  1px solid rgba(0,240,255,0.5)
激活边框：  1px solid rgba(0,240,255,0.8)
面板分隔：  1px solid rgba(0,240,255,0.08)
```

## 十二、字体

| 用途 | 字体 | 大小 | 字重 | 颜色 |
|------|------|------|------|------|
| 游戏标题 | Inter / SF Pro Display | 44px | Bold | #ffffff + 发光 |
| 页面标题 | Inter | 20px | SemiBold | #ffffff |
| 导航文字 | Inter | 14px | Medium | var(--text) |
| HUD数值 | JetBrains Mono / 等宽 | 16px | Bold | #ffffff |
| HUD标签 | Inter | 11px | Regular | rgba(255,255,255,0.4) |
| 物品名称 | Inter | 12px | Medium | #ffffff |
| 物品价值 | JetBrains Mono | 11px | Bold | #00f0ff |
| 提示文字 | Inter | 12px | Regular | rgba(255,255,255,0.4) |
| 按钮文字 | Inter | 16px | SemiBold | 随状态 |

**数值优先使用等宽字体**，确保 HP、弹药、金币等数字严格对齐。
