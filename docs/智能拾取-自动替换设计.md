# 智能拾取 — 背包满时自动替换

## 功能定位

玩家背包满了，打开一个掉落物箱子时，系统自动对比背包和箱子里的物品价值，**丢弃低价值的腾出格子，拾取高价值的进背包**。本质是：一键"扔垃圾，拿好货"。

---

## 核心逻辑

```
玩家已满背包打开箱子
        ↓
系统扫描背包中所有物品，按价值密度排序（价值/格），从低到高
系统扫描箱子中所有物品，按价值密度排序，从高到低
        ↓
逐对比较：
  如果 箱子中最高价值的物品 > 背包中最低价值的物品
    → 交换：背包最低价值物品 → 丢进箱子
    → 交换：箱子最高价值物品 → 拾取进背包
    → 继续下一对比较
  否则
    → 停止（剩下的物品不如你已有的，不换）
        ↓
屏幕显示：
  ┌──────────────────────────────┐
  │  智能拾取完成                 │
  │  ✓ 金链子（220） ← 已拾取   │
  │  ✗ 破布（10）   → 已丢弃    │
  │  ──────────────────────      │
  │  背包净值变化：+210 金币     │
  └──────────────────────────────┘
```

## 交互流程

```
玩家打开箱子 → 屏幕出现拾取界面
        ↓
界面默认显示普通拾取列表（和现在一样）
        ↓

        ↓

  1. 玩家看到背包里的破布/废铁被自动放进箱子
  2. 箱子里的金链子/钻石自动进入背包
  3. 屏幕中央弹出替换结果摘要
        ↓

```

也可以让智能拾取在打开箱子时**自动触发**（一步到位）：

```
玩家打开箱子
        ↓
系统检测背包已满 → 自动执行替换逻辑
        ↓
直接显示替换结果
        ↓
玩家确认后关闭
```

## 价值对比规则

### 价值密度优先

对比的不是单格价值，而是**价值密度 = 物品价值 ÷ 占用格数**。因为一个大件占 4 格但价值 800，不如 4 个 1 格 250 的小件。

```
背包：金佛像（1200，占2格）→ 密度 600/格
箱子：破布（10，占1格）    → 密度 10/格
→ 不换，金佛像比破布值钱得多

背包：破布（10，占1格）     → 密度 10/格  
箱子：金戒指（100，占1格）  → 密度 100/格
→ 换，金戒指密度远高于破布
```

### 不触发的条件

- 背包未满 —— 正常拾取，不触发替换逻辑
- 箱子里最好的物品不如背包里最差的 —— 不换，弹提示"没有更好的物品"
- 箱子里有空间但背包物品放不进去（格子形状不匹配） —— 跳过该物品

## 实现代码框架

```lua
function SmartSwap(backpackItems, chestItems)
    -- 背包物品按价值密度从低到高排序（最先丢最垃圾的）
    table.sort(backpackItems, function(a, b)
        return a.value / a.slots < b.value / b.slots
    end)

    -- 箱子物品按价值密度从高到低排序（先拿最好的）
    table.sort(chestItems, function(a, b)
        return b.value / b.slots > a.value / a.slots
    end)

    local swapped = {}
    local backpackDropped = {}
    local chestTaken = {}

    -- 逐对尝试交换
    local bi = 1  -- 背包索引（从最垃圾的开始）
    local ci = 1  -- 箱子索引（从最好的开始）

    while bi <= #backpackItems and ci <= #chestItems do
        local bpItem = backpackItems[bi]
        local chestItem = chestItems[ci]

        -- 如果箱子里最好的还不如背包里最垃圾的，停止
        if chestItem.value / chestItem.slots <= bpItem.value / bpItem.slots then
            break
        end

        -- 检查格子形状是否兼容（这是个简化，实际需按背包系统检查）
        if CanFitInChest(bpItem, chestItems) and CanFitInBackpack(chestItem, backpackItems) then
            -- 执行交换
            table.insert(backpackDropped, bpItem)
            table.insert(chestTaken, chestItem)
            bi = bi + 1
            ci = ci + 1
        else
            -- 格子不匹配，跳过箱子这个物品
            ci = ci + 1
        end
    end

    -- 执行实际移动
    for _, item in ipairs(backpackDropped) do
        RemoveFromBackpack(item)
        AddToChest(item)
    end
    for _, item in ipairs(chestTaken) do
        RemoveFromChest(item)
        AddToBackpack(item)
    end

    return swapped, backpackDropped, chestTaken
end
```
