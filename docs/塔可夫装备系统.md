# 塔可夫风格装备系统（极简版）

> 参照《逃离塔科夫》的全身装备栏：每个部位独立装备。
> 无胸挂、无口袋，只有核心装备槽 + 背包网格。

## 四、玩家装备数据模型

```javascript
class PlayerEquipment {
  constructor() {
    this.slots = {
      helmet: null,
      armor: null,
      backpack: null,
      primary: null,
      secondary: null,
    };
    // 背包网格（初始为口袋大小 2×2）
    this.backpackGrid = new GridInventory(2, 2);
  }

  // 装备物品
  equip(slotId, item) {
    if (this.slots[slotId]) this.unequip(slotId);
    this.slots[slotId] = item;
    // 如果装备的是背包，替换背包网格
    if (slotId === 'backpack' && item.internalGrid) {
      const newGrid = new GridInventory(item.internalGrid.w, item.internalGrid.h);
      // 尝试迁移旧物品
      for (const oldItem of this.backpackGrid.items) {
        const space = newGrid.findSpace(oldItem.w, oldItem.h);
        if (space) newGrid.placeItem(oldItem, space.x, space.y);
      }
      this.backpackGrid = newGrid;
    }
    return true;
  }

  // 卸下装备
  unequip(slotId) {
    const item = this.slots[slotId];
    if (!item) return null;
    this.slots[slotId] = null;
    if (slotId === 'backpack') {
      const contents = [...this.backpackGrid.items];
      this.backpackGrid = new GridInventory(2, 2);
      return { item, contents };
    }
    return { item, contents: [] };
  }

  getTotalArmor() {
    let armor = 0;
    if (this.slots.helmet) armor += this.slots.helmet.armor || 0;
    if (this.slots.armor) armor += this.slots.armor.armor || 0;
    return armor;
  }

  getTotalCapacity() {
    return this.backpackGrid.width * this.backpackGrid.height;
  }

  // 阵亡：全部丢失
  onDeath() {
    for (const k of Object.keys(this.slots)) this.slots[k] = null;
    this.backpackGrid = new GridInventory(2, 2);
    return [];
  }

  // 存活：全部物品返回
  onExtract() {
    const all = [];
    for (const k of Object.keys(this.slots)) {
      if (this.slots[k]) all.push(this.slots[k]);
    }
    for (const item of this.backpackGrid.items) all.push(item);
    return all;
  }

  autoAddItem(item) {
    const space = this.backpackGrid.findSpace(item.w, item.h);
    if (space) {
      this.backpackGrid.placeItem(item, space.x, space.y);
      return true;
    }
    return false;
  }
}
```

## 五、与旧系统的关系

| 旧系统 | 新系统 |
|--------|--------|
| `player.inventory[]` 列表 | 删除，由 `equip.backpackGrid` 替代 |
| `player.ammo` / `player.maxAmmo` | 删除，弹药作为物品放入背包网格 |
| `player.armor` / `player.helmet` | `equip.slots.armor` / `.helmet` |
| `player.weapon` | `equip.slots.primary` |
| `player.rig` / `player.backpack` | 只有 `equip.slots.backpack` |
| `getMaxInventory()` | `equip.getTotalCapacity()` |
| `tryAddToInventory()` | `equip.autoAddItem()` |
