## 核心玩法概述

玩家扮演一只持枪的猫，闯入 20 层随机生成地牢。核心循环：**搜刮 → 战斗 → 撤离**。  
成功撤离：背包物品存入仓库。阵亡：本次搜刮全部丢失。

设计参考：
- 撤离搜刮：《逃离塔科夫》
- Build 天赋：《杀戮尖塔》《土豆兄弟》《哈迪斯》《枪火重生》
- 背包系统：《逃离塔科夫》（网格背包）

## 游戏状态机

```
STATE 取值及流转：

menu → hub → loadout → pick → playing → extract_choice → win
                                  ↓              ↓
                                reward      (继续) → playing
                                  ↓
                             playing (下一层)
                                  ↓
                              gameover
```

| 状态 | 说明 |
|------|------|
| `menu` | 主菜单（开始/音量/玩法介绍） |
| `hub` | Hub 主界面（进入准备/仓库/技能展示） |
| `loadout` | 战前准备（从仓库取装备、商人买卖） |
| `pick` | 出发前三选一初始天赋 |
| `playing` | 地牢战斗主循环 |
| `paused` | 暂停菜单 |
| `reward` | 层间三选一 Build 奖励 |
| `extract_choice` | 撤离/继续深入选择（1~15层） |
| `deep_confirm` | 深入确认对话框 |
| `gameover` | 死亡结算 |
| `win` | 胜利结算（撤离成功或通关第20层） |

## Build 天赋系统

### 6 大流派

| 流派 | 键名 | 方向 |
|------|------|------|
| 火力压制 🔥 | firepower | 伤害/弹射/爆炸/弹药 |
| 生存龟壳 🛡️ | tank | 护甲/回血/减伤/坦度 |
| 机动游击 💨 | agile | 移速/换弹/闪避/瞬移 |
| 暴击一击 🎯 | crit | 暴击率/暴击伤害/必暴 |
| 特效失控 ✨ | chaos | 弹射/燃烧/连锁/混沌 |
| 一刀流 🔪 | knife | 近战范围/伤害/AOE |

### 阈值机制

每个流派在投入 3/5/7 点时解锁里程碑效果：

**firepower 示例**：
- 3点：过热枪管 — 连续命中同一敌人5次触发额外爆炸
- 5点：弹射子弹 — 子弹命中后弹射到附近另一个敌人
- 7点：无限火力 — 击杀后3秒内不消耗弹药

### 三选一规则

- 每层通关后生成 3 张天赋卡（默认，可被"四选一牌"增至4张）
- 支持刷新（reroll），次数有限
- 保底机制：连续未出高稀有度时提升概率

## 关键设计决策备注

1. **死亡惩罚严厉**：阵亡丢失全部本局物品，只有成功撤离才保存
2. **层数越深，风险越高回报越大**：稀有度权重随层数大幅倾斜
3. **16~19层不可撤退**：制造紧张感，逼迫玩家决策是否在15层前撤离
4. **Boss 宝箱奖励丰厚**：击杀 Boss 后生成多个高品质宝箱
5. **Build 系统增加重玩性**：每局天赋组合不同，鼓励尝试不同流派
6. **搜索有前摇**：物品逐一加载，增加搜刮时的紧张感（可被敌人打断）
7. **视野限制**：有限 FOV 增加探索紧张感，撤离点需先发现才显示导航
